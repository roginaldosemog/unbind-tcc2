# UNBIND API

API em Python referente ao projeto UNBIND

## Requisitos

Para rodar o projeto você deve ter instalado:
- Python 3.13
- FastAPI (https://fastapi.tiangolo.com/#installation)

## Instalação

```bash
.\.venv\Scripts\activate
fastapi dev main.py
```

Ao finalizar:
```bash
deactivate
```
